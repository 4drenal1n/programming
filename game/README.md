# Бой с монстром (Clicker Game)

Простая кликер-игра с графическим интерфейсом на Tkinter.  
Игрок атакует монстра, зарабатывает монеты и покупает зелья для восстановления здоровья.  
С каждым новым уровнем монстр становится сильнее.

## Требования

- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)

## Установка и запуск

### 1. Клонирование репозитория
```bash

git clone <url-репозитория>
cd <директория-проекта>
```
### 2. Создание и активация виртуального окружения
```bash

python -m venv venv
```
#### Активация виртуального окружения:

Windows: venv\Scripts\activate

Linux / macOS: source venv/bin/activate

### 3. Запуск игры
```bash
python clicker_game.py
```
